docker-maldet
===========

This is docker project for check files on malware files using maldet.


Dependecies
===========
You have to install Docker, make in your host operating system.


How it's work
===========

1. Pull image:

    `` docker pull vukor/docker-maldet ``

2. Go to directory with your web files

3. Check your files:

    `` make check ``

4. See reports:

    `` make reports ``

5. See help:

    `` make help ``
